# Church App

## Overview

The Church App is a web application built using Python and Flask that serves as a comprehensive platform for church-related activities. It includes features such as a church directory, live stream integration, event calendar, prayer request submissions, announcements, donation forms, and a first-time guest section.

## Features

- **Church Directory**: A list of churches with their respective locations.
- **Live Streams**: Integration of live-streamed services and recorded videos.
- **Event Calendar**: Displays upcoming church events.
- **Prayer Request Submission**: Allows users to submit prayer requests which are stored for later retrieval.
- **Announcements**: A section for important church announcements.
- **Donation Form**: Facilitates online donations to the church.
- **First-Time Guests**: A dedicated section to welcome and provide information for first-time guests.

## Technologies Used

- **Backend**: Python, Flask
- **Frontend**: HTML, CSS, JavaScript
- **Templates**: Jinja2 (Flask's templating engine)
- **Deployment**: Flask's built-in development server
